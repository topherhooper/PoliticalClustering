# PoliticalClustering

## Getting Started

### Setup (Install docker and stuff)
```bash
bash bash_lib/setup.sh
```

### Put API Key in api_data_gov.key
```bash
echo <you api key> > api_data_gov.key
```



### Start Jupyter Service
```bash
bash bash_lib/shell.sh
```

### Naviate to:
[127.0.0.1:10000](127.0.0.1:10000)
